# 使用說明書

## 詳細使用步驟

### 1. 準備音頻文件

支援的音頻格式：
- MP3 (.mp3)
- WAV (.wav)
- M4A (.m4a)
- FLAC (.flac)
- OGG (.ogg)
- WMA (.wma)

### 2. 選擇音頻來源

**選項 A：使用 Google Drive**
- 將音頻文件上傳到 Google Drive 的 `Conv2Sub` 文件夾
- 工具會自動列出該文件夾中的所有音頻文件
- 從下拉選單選擇要處理的文件

**選項 B：手動上傳**
- 選擇「手動上傳文件」
- 點擊上傳按鈕選擇本地音頻文件
- 文件會上傳到 Colab 臨時空間

### 3. 配置參數

**語言選擇**
- 下拉選單選擇目標語言
- 檔案後綴會自動對應（如 zh-TW、en-US 等）

**初始提示詞**
- 可選擇已保存的提示詞
- 或輸入自定義提示詞並保存
- 提示詞可提高特定場景的識別準確率

**模型選擇**
- 根據 GPU 記憶體選擇合適模型
- Large-v3-Turbo 是推薦選項

### 4. 開始處理

點擊「開始生成字幕」按鈕，處理過程包括：
1. 人聲片段提取（使用 FFmpeg）
2. 語音識別（WhisperX）
3. 逐字對齊
4. 多格式字幕生成

### 5. 獲取結果

**使用 Google Drive 時：**
- 字幕文件打包為 ZIP 保存在 `Conv2Sub/subtitle_output` 文件夾
- 可直接在 Drive 中下載使用

**手動上傳時：**
- 字幕文件保存在臨時目錄
- 提供每個文件的下載鏈接
- 提供打包 ZIP 下載鏈接
