# 使用範例

## 範例 1：中文 podcast 轉字幕

**輸入文件**：`podcast_episode_1.mp3`（45分鐘）
**配置**：
- 語言：繁體中文
- 模型：Large-v3-Turbo
- 提示詞：科技類 podcast 討論人工智慧

**處理時間**：約 8 分鐘
**輸出文件**：
- `podcast_episode_1.zh-TW.srt`
- `podcast_episode_1.zh-TW.vtt`
- `podcast_episode_1.zh-TW.tsv`
- `podcast_episode_1.zh-TW.txt`

## 範例 2：英語教學視頻

**輸入文件**：`english_lesson.m4a`（15分鐘）
**配置**：
- 語言：英文 (美國)
- 模型：Small（快速測試）
- 提示詞：English learning for beginners

**處理時間**：約 2 分鐘
**輸出文件**：
- `english_lesson.en-US.srt`
- `english_lesson.en-US.vtt`
- `english_lesson.en-US.tsv`
- `english_lesson.en-US.txt`

## 範例 3：日語動畫對話

**輸入文件**：`anime_dialogue.wav`（5分鐘）
**配置**：
- 語言：日文
- 模型：Large-v3（最高精度）
- 提示詞：アニメの会話、日常会話

**處理時間**：約 3 分鐘
**輸出文件**：
- `anime_dialogue.ja.srt`
- `anime_dialogue.ja.vtt`
- `anime_dialogue.ja.tsv`
- `anime_dialogue.ja.txt`
